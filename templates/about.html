Background

In the previous testing process, developers submitted test applications to testers. They can only fill in the test intention, branch and other information through the confidence template, and then email and copy related colleagues. The steps are more complicated and it is not conducive to information statistics archiving. The Application System is based on this requirement.

Design

Due to the simple requirements, in order to facilitate development and maintenance, TAS is developed using the lightweight web framework Flask + bulma, the database is MySQL, and Gunicorn + Nginx is deployed. Please log in with a domain account to create a test request.
